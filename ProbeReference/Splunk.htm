Splunk > Layer 2 CSV Output

Switch Layer 2 CSV Output

This probe periodically scans InterMapper's Layer 2 database for information collected from switch devices, and writes it into a CSV file. The InterMapper App for Splunk (http://www.intermapper.com/products/Splunk-App/) then retrieves this data for use within Splunk.

If the process fails for any reason, the device is set to Warning.

To use this probe:

1) Create a device on any map and set the device's address to "localhost".

2) The default poll interval is 300 seconds (five minutes). Lower this interval if your Layer2 discovery runs more frequently.

Note:

One and only one device using this probe should exist on an InterMapper server. Running multiple instances of this probe uses InterMapper server resources unneccessarily, with no benefit to the InterMapper App for Splunk.
